5) Test the stationarity using the KPSS¶

Test for Stationarity Using KPSS¶

The output of the KPSS test contains 4 things:

  • The KPSS statistic

  • p-value

  • Number of lags used by the test

  • Critical values

Reference - normaltest

resid11 = arima11.resid

norm_test = stats.normaltest(resid11)
print(f"Normality test statistic: {norm_test[0]}")
print(f"p-value: {norm_test[1]}")

Normality test statistic: 0.06446943472258876
p-value: 0.9682792834463669

This function (normaltest()) tests the null hypothesis that a sample comes from a normal distribution. The p-value is 0.9683, which is much greater than 0.05. p-value > 0.05 means we FAIL to reject the null hypothesis. The data comes from a normal distribution, showing that my residuals are normally distributed. This shows that my model is good.
